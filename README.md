# ValClabs

Sitio web responsive de ValClabs listo para GitHub Pages.

## Publicación

1. Crea un repositorio en GitHub
2. Sube todos los archivos de esta carpeta, no la carpeta contenedora
3. Ve a Settings → Pages
4. Selecciona Deploy from a branch
5. Elige `main` y `/root`

## Antes de publicar

Reemplaza `TU-USUARIO` y `TU-REPOSITORIO` en:

- index.html
- robots.txt
- sitemap.xml
