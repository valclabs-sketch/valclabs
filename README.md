- assets/

## Publicar en GitHub Pages

1. Crea un repositorio nuevo o elimina los archivos antiguos del repositorio actual
2. Sube directamente todos los archivos y la carpeta `assets`
3. Ve a `Settings`
4. Abre `Pages`
5. En `Build and deployment`, selecciona `Deploy from a branch`
6. Selecciona la rama `main` y la carpeta `/root`
7. Guarda los cambios

## Reemplazos antes de publicar

Cambia `TU-USUARIO` y `TU-REPOSITORIO` en:

- index.html
- robots.txt
- sitemap.xml

Ejemplo:

`https://valclabs-sketch.github.io/valclabs/`
